# Code challenge Senirop

Solutions proposed to the exercises challenge

## Requirements

- Node js >= 20
- npm

## Instrucctions

1. Clone this repository

```bash
git clone https://github.com/devjaes/code-challenge-sp.git
```

2. Install the dependencies

```bash
npm i
```

### To Run the tests

```bash
npm run test
```

### To Run exercises solutions

```bash
npm run start
```
